# Сохранение информации о дисках

Для сохранения информации о структуре дисков использовались стандартные утилиты Linux.

Примеры команд:

```bash
lsblk /dev/sdX
```

```bash
sudo parted /dev/sdX print
```

```bash
sudo fdisk -l /dev/sdX
```

Вывод этих команд сохранён в каталоге:

```
results/
```
