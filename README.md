# MBR_partitions
# Linux Disk Lab — MBR + 50 Logical Partitions (Debian 13)

Подготовлена виртуальная машина Debian 13 с дополнительным диском, размеченным в формате **MBR (DOS)**.  
На диске создан **1 extended-раздел** и **50 logical-разделов** (sdb5–sdb54), все отформатированы в **ext4** и автоматически монтируются при загрузке через `/etc/fstab`.

### 1) Окружение
- VirtualBox + Debian 13
- Дополнительный диск: `/dev/sdb` (~20GiB)

### 2) Разметка диска (MBR)
- Таблица разделов: **MBR (Disklabel type: dos)**
- `sdb1` — **extended**
- `sdb5 … sdb54` — **50 logical разделов**

### 3) Файловая система
- Все logical-разделы отформатированы в **ext4**

### 4) Монтирование и автоподключение
- Точки монтирования: `/mnt/data/sdb5 … /mnt/data/sdb54`
- Автоматическое монтирование настроено в `/etc/fstab` через **UUID**
- Проверено после `reboot`: все разделы остаются смонтированными
