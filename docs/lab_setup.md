# Environment Setup

Работа выполнялась в виртуальных машинах Linux.

Используемые системы:

* Debian 13
* Kali Linux

В каждую виртуальную машину были добавлены дополнительные диски для работы с таблицами разделов.

Пример проверки дисков:

```bash
lsblk
```

Дополнительные диски использовались только для работы с разделами и не содержали операционную систему.
