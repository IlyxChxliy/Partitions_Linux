# MBR Partition Setup

Для работы с таблицей разделов MBR использовалась утилита `fdisk`.

Сначала создавалась таблица разделов:

```bash
fdisk /dev/sdc
```

Затем создавался extended-раздел, занимающий весь диск.

После этого создавались logical-разделы.

Часть разделов создавалась вручную, часть с помощью скриптов.

Пример команды создания раздела:

```bash
n
l
+50M
```

Также использовался скрипт:

```
scripts/mbr_create_random.sh
```

который создаёт logical-разделы случайного размера.

Информация о разделах сохранялась с помощью команд:

```bash
fdisk -l
lsblk
parted print
```

Результаты сохранены в каталоге:

```
results/*/mbr
```
