# GPT Partition Setup

Для работы с GPT использовалась утилита `parted`.

Создание таблицы разделов:

```bash
parted /dev/sdb mklabel gpt
```

После этого создавались разделы.

Для автоматизации использовался скрипт:

```
scripts/gpt_create_128.sh
```

Скрипт создаёт разделы случайного размера.

При попытке создать больше разделов появляется ошибка:

```
Error: Too many primary partitions
```

Информация о структуре диска сохранялась с помощью команд:

```bash
lsblk
parted print
```

Результаты сохранены в каталоге:

```
results/*/gpt
```

## Notes

По умолчанию таблица разделов GPT создаётся с ограничением в 128 записей.

При попытке создать больше разделов утилита `parted` возвращает ошибку:

```
Error: Too many primary partitions
```

Это ограничение связано с размером таблицы разделов, создаваемой по умолчанию.

При необходимости количество разделов может быть увеличено с помощью специализированных утилит, например `sgdisk`, которые позволяют изменить размер таблицы разделов GPT.
